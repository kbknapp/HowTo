# Sublime Text 3
Cheatsheet used to help with the most common and useful features of Sublime Text 3
## Keyboard Shortcuts
```⌘⇧P``` Command Palet

```⌘P``` Goto anything

```⌘B⌘K``` Toggle Sidebar

```⌥⇧N``` Advanced New File

## Prefrences
Always change User settings

```Sublime Text -> Preferences -> Settings-User```
